## ЧЕКОР 1 — Постави на GitHub

```bash
# 1. Создај нов репозиториум на github.com (јавен!)
# 2. Иницијализирај git во папката
cd notes-app
git init
git add .
git commit -m "Initial commit: Notes App"

# 3. Поврзи со GitHub и пушни
git remote add origin https://github.com/ТВОЕ_IME/notes-app.git
git branch -M main
git push -u origin main
```

---

## ЧЕКОР 2 — Тестирај локално со Docker Compose

```bash
# Стартувај ги сите сервиси
docker compose up --build

# Отвори во браузер:
# http://localhost  ← frontend
# http://localhost:3000/health  ← backend health check

# Запре:
docker compose down
```

---

## ЧЕКОР 3 — Постави DockerHub Secrets на GitHub

1. Оди на GitHub → твоиот репозиториум → **Settings → Secrets and variables → Actions**
2. Додај два secrets:
   - `DOCKERHUB_USERNAME` = твоето DockerHub корисничко ime
   - `DOCKERHUB_TOKEN` = DockerHub Access Token (генерирај го на dockerhub.com → Account Settings → Security)

---

## ЧЕКОР 4 — Активирај CI/CD Pipeline

```bash
# Направи промена и пушни — GitHub Actions ќе работи автоматски
git add .
git commit -m "Trigger CI/CD pipeline"
git push
```

Оди на GitHub → **Actions** и следи го pipeline-от. По успешното завршување, Docker images ќе бидат на DockerHub.

## ЧЕКОР 5 — Замени го корисничкото ime во k8s манифестите

Во фајловите `k8s/05-backend-deployment.yaml` и `k8s/07-frontend-deployment.yaml` замени:
```
YOUR_DOCKERHUB_USERNAME  →  твоето DockerHub корисничко ime
```

## ЧЕКОР 6 — Поставување на Kubernetes (Minikube)

```bash
# Инсталирај Minikube (ако немаш)
# https://minikube.sigs.k8s.io/docs/start/

# Стартувај Minikube со Ingress
minikube start
minikube addons enable ingress

# Примени ги сите манифести (по ред)
kubectl apply -f k8s/

# Провери статус
kubectl get all -n notes-app

# Чекај да се стартуваат сите pods (може 1-2 минути)
kubectl wait --for=condition=ready pod --all -n notes-app --timeout=120s

# Земи ја IP адресата на Minikube
minikube ip
# На пример: 192.168.49.2
```

### Додај host запис (за notes.local домен)

```bash
# macOS/Linux — уреди /etc/hosts:
sudo echo "$(minikube ip)  notes.local" >> /etc/hosts

# Windows — уреди C:\Windows\System32\drivers\etc\hosts:
# Додај: 192.168.49.2  notes.local
```

Потоа отвори: **http://notes.local**

## Корисни команди

```bash
# Листа на сите resources во namespace-от
kubectl get all -n notes-app

# Логови на backend
kubectl logs -l app=backend -n notes-app

# Логови на postgres
kubectl logs -l app=postgres -n notes-app

# Избриши сè и почни одново
kubectl delete namespace notes-app
kubectl apply -f k8s/

# Провери Ingress
kubectl get ingress -n notes-app
```
